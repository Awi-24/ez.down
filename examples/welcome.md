# Welcome to ez.down

This is a **WYSIWYG Markdown editor**. You edit the _formatted_ text directly —
select any text to reveal the formatting toolbar, or type `/` to insert a block.

## GFM features

- [x] Task lists
- [ ] To-do items
- Regular lists
  - with nesting
